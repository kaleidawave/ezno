### Classes


#### Class methods

```ts
class X {
	constructor(value) {
		this.value = value
	}

	getObject(b) {
		return { a: this.value, b }
	}
}

const x = new X(4)
x.getObject(2) satisfies string
```

- Expected string found {"a": 4, "b": 2, }

#### Resolving value by property on dependent


- Expected "a" | "b" | "c" found "d"
